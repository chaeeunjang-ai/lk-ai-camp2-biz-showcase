# LK AI Camp 2기 — 사업팀 쇼케이스

사업팀 AI 캠프 2기 참가자들의 쇼케이스 자료를 모으는 곳입니다.

---

## 제출 방법

Claude Code에 아래 프롬프트를 복사해서 붙여넣으세요.

```
https://github.com/arin-ship-it/lk-ai-camp2-biz-showcase 레포를 clone하고 해당 폴더로 이동한 뒤, /template/github_showcase_pr_prompt.md 파일의 지시사항을 따라서 작업을 진행해줘.
```

---

## 진행 과정

1. 레포 클론
2. 내 브랜치 만들기 (`영문이름/showcase`)
3. `submissions/영문이름/` 폴더에 템플릿 복사
4. SHOWCASE.md 작성 (①~⑥ 항목 채우기)
5. 커밋 & PR 올리기
6. [전체 PR 목록](https://github.com/arin-ship-it/lk-ai-camp2-biz-showcase/pulls)에서 다른 팀원 제출물 보기
